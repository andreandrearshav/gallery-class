# 📸 Gallery Kelas

Website gallery kelas berbasis **Next.js** untuk menampilkan dokumentasi kegiatan dan anggota kelas dengan tampilan modern (Neubrutalism style).

---

## 🚀 Fitur Utama

- 📷 Gallery foto kegiatan kelas
- 👥 Data anggota kelas (struktur)
- ⚡ UI (Neubrutalism design)

---

## 🛠️ Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Language**: JavaScript / TypeScript
- **Deployment**: Vercel
